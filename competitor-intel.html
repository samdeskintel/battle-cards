Setup instructions
The API key is held in memory only — never written to storage or sent anywhere except directly to Anthropic.

1. Grab an Anthropic API key from console.anthropic.com — create one under a shared team account so the whole sales team uses the same key.
2. Share it internally — a pinned Slack message to the sales channel works fine for an internal tool.
3. Reps paste it in when they open the page — takes 5 seconds. It stays active for the session.

The anthropic-dangerous-direct-browser-access header is required for direct browser-to-API calls — it's Anthropic's own mechanism for this use case (no backend needed). The key is never logged or stored.
